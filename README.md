# 📊 Sales Data Analysis

This project demonstrates basic data analysis and visualization using Python.

## 🔧 Tools Used
- Python
- Pandas
- Matplotlib

## 📁 Dataset
A sample sales dataset containing product, region, and sales data.

## 📊 Key Analysis
- Sales by Product
- Sales by Region
- Monthly Sales Trend

## 📈 Visualizations
- Bar Chart (Product vs Sales)
- Line Chart (Monthly Trend)

## 🚀 Outcome
Extracted insights from raw data and visualized trends to support decision-making.
